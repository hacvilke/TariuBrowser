❓ Frequently Asked Questions

How do I change the homepage?
Click ☰ → Tools → Settings. Enter any URL in the Homepage field. Leave blank or type home to use the built-in homepage with search.
How do I change the search engine?
Click ☰ → Tools → Settings. In the Search Engine URL field, paste a URL with %s where the search query goes.

Examples:
• Google: https://www.google.com/search?q=%s
• DuckDuckGo: https://duckduckgo.com/?q=%s
How do I bookmark a page?
Click the ⭐ button in the toolbar, or press Ctrl+D. The star turns gold when bookmarked. View all bookmarks at ☰ → Bookmarks → Show Bookmarks (Ctrl+B).
How do I view my browsing history?
Click ☰ → History → Show History (Ctrl+H). You can search through your history and double-click any entry to revisit it. Clear history from the same menu.
How do I open Developer Tools?
Press F12 or click ☰ → View → Developer Tools. This opens full Chrome DevTools for inspecting elements, debugging JavaScript, and monitoring network requests.
How do I clear browsing data?
Click ☰ → Tools → Clear Browsing Data. This removes cookies, cache, and local storage. Your bookmarks and settings are not affected.
How do I manage downloads?
Click the 📥 downloads button in the toolbar, or press Ctrl+J. This opens the Download Manager showing progress and status of all downloads.
How do I update the browser?
The browser automatically checks for updates on startup. If a new version is available, you'll see a popup. You can also manually check at ☰ → Help → Check for Updates.
What keyboard shortcuts are available?
Ctrl+T — New Tab
Ctrl+W — Close Tab
Ctrl+L — Focus URL Bar
Ctrl+D — Bookmark Page
Ctrl+F — Find in Page
Ctrl+H — History
Ctrl+B — Bookmarks
Ctrl+J — Downloads
F5 / Ctrl+R — Reload
F11 — Fullscreen
F12 — DevTools
Alt+← — Back
Alt+→ — Forward
Can I use a custom User Agent?
Yes! Go to ☰ → Tools → Settings and enter a custom User Agent string. Leave blank to use the default Chromium user agent.
