## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Built-in Homepage** | Beautiful search page with quick links — no server needed |
| 📑 **Multi-Tab Browsing** | Chrome-style tabs with full keyboard shortcuts |
| ⭐ **Bookmarks** | Save and organize your favorite pages |
| 📜 **History** | Searchable browsing history, stored locally |
| 🛠 **DevTools** | Full Chrome DevTools — press F12 |
| 📥 **Downloads** | Built-in download manager with progress tracking |
| 🔍 **Find in Page** | Quick text search with Ctrl+F |
| 🎨 **Retro-Modern UI** | Warm beveled design with SVG icons |
| 📊 **Live Telemetry** | Real-time CPU & RAM in status bar |
| 🔒 **Privacy First** | Direct connections, no proxy, no tracking |

## ⌨️ <a name="shortcuts"></a>Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + T` | New Tab |
| `Ctrl + W` | Close Tab |
| `Ctrl + L` | Focus URL Bar |
| `Ctrl + D` | Bookmark Page |
| `Ctrl + F` | Find in Page |
| `Ctrl + H` | Show History |
| `Ctrl + B` | Show Bookmarks |
| `Ctrl + J` | Show Downloads |
| `Ctrl + Tab` | Next Tab |
| `Ctrl + Shift + Tab` | Previous Tab |
| `F5` / `Ctrl + R` | Reload Page |
| `F11` | Fullscreen |
| `F12` | Developer Tools |
| `Alt + ←` | Back |
| `Alt + →` | Forward |
| `Ctrl + Q` | Quit |

## 🛠️ Tech Stack

- **UI Framework:** PyQt6
- **Browser Engine:** QtWebEngine (Chromium)
- **Icons:** SVG (vector)
- **Packaging:** PyInstaller
- **Installer:** Inno Setup
