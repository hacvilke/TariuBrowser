# Changelog

## [3.0.0] - 2026-06-24

### Added
- Chrome-inspired tab layout with custom tab bar
- Built-in homepage with search and quick links
- SVG icons for all navigation buttons
- Hamburger menu replacing old menu bar
- Download manager with progress tracking
- Live CPU/RAM telemetry in status bar
- Bookmarks and history with search
- Keyboard shortcuts (Ctrl+T/W/L/D/F/H/B/J)
- Context menu with page source, save, inspect
- Settings dialog (homepage, search engine, user agent)
- Clear browsing data option
- New window support
- Persistent window geometry and settings

### Changed
- Single retro-modern theme (removed theme toggle)
- Tabs now below URL bar (Chrome-style)
- Rounded tab corners with gradient backgrounds
- Circular navigation buttons with hover effects
- Improved stylesheet for all dialogs and menus

### Fixed
- Proxy errors (direct connections now)
- QPolygonF compatibility with PyQt6
- CPU telemetry seeding for accurate first reading
