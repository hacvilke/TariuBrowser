# Contributing

This is a closed-source project. We accept:

- ✅ Bug reports via [Issues](https://github.com/YOUR_USER/tariu-browser/issues)
- ✅ Feature requests via [Issues](https://github.com/YOUR_USER/tariu-browser/issues)

We do NOT accept:

- ❌ Pull requests with code changes
- ❌ Source code contributions

## Bug Reports

Use the bug report template and include:
- OS version
- Steps to reproduce
- Expected vs actual behavior
- Screenshots if applicable

## Feature Requests

Describe what you want and why it would be useful.
